
# Schema Drift Detector

This PoC compares two schema snapshots stored as JSON files, detects drift,
uses a placeholder AI analysis module, and displays results.

## Run

pip install -r requirements.txt

### CLI
python app.py

### Streamlit Dashboard
streamlit run dashboard.py
